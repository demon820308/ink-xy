# 🖋️ ink-xY Novel Studio — 智能协同写作空间

> 🌌 **ink-xY Novel Studio** 是一款专为小说创作者与文学家打造的 **非线性沉浸式智能协同写作空间**。基于 Next.js 15 Web 技术与 Electron Standalone 独立桌面客户端完美融合，为作家提供温润如纸的视觉底色、高雅文学排版、极简非线性大纲管理器，以及协同写作 AI 写作伴侣。

---

## 🎨 核心闪光点

* 🖥️ **全平台独立文学客户端** — 原生支持 Windows 与 macOS 独立客户端，温润纸张黄与深邃静心绿双色主题，为写作者筑起专注防线。
* 🌳 **非线性大纲与创作目录** — 侧边栏 elevated 创作目录树，大纲、章节、人物卡片、世界观设定一目了然，支持分标签页沉浸编辑。
* 📝 **Zen 沉浸创作编辑器** — 极致纯净的 Markdown 创作编辑器，支持大行距 (1.8)、衬线体高雅文学排版，底部内置实时中文字数与行数统计。
* 💾 **REST 级毫秒级自动保存** — 内置 1500ms 智能去抖（Debounce）自动保存机制，无缝将您的创作存盘，让您专注于灵感。
* 🔮 **AI 写作伴侣 (Co-writers)** — 右侧可折叠的 AI 协作伴侣面板，支持智能头脑风暴、角色润色与剧情推演，深度保护写作者的专注力。
* 🤫 **纯净创作环境** — 全面净化调试日志与极客控制台，将技术思维彻底留给代码，把灵感与故事留给墨水。

---

## 🧭 如何启动与运行

### 1. 运行 Next.js Web 界面
```bash
# 安装依赖
npm install

# 启动 Web 服务器 (端口 30142)
npm run dev
```

### 2. 启动 Standalone 桌面客户端
```bash
# 启动 Standalone 文学桌面版
npm run electron:dev
```

---

## 🔒 独立运行与隐私保护
所有项目设置、写作历史会话、创作配置文件皆完全隔离，存储在 `~/.ink/agent/` 下，实现完全独立与本地存盘，保障您的创作隐私安全。
